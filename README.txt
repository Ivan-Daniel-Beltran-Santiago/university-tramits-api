Node.js backend for managing student service requests, including claim submissions and scalable to support additional procedures.

Technologies.

- Node.js
- AngularJS
- Microsoft SQL Server

Features:

- RESTful endpoints
- Modular routing
- Scalable design for multiple request types

Getting Started:

git clone https://github.com/Ivan-Daniel-Beltran-Santiago/diagramasions2_la-venganza.git
cd diagramasions2_la-venganza
npm install
npm start

Notes:

Originally developed for a single use-case, but designed to allow scaling.
